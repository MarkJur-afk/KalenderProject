Microsoft Project – Diagrammide loomine

Juhend: kuidas luua ja kohandada Project Overview aruannet

  • Uue kalendri loomine
  • Arvutusvälja lisamine
  • Diagrammide loomine

1. Ava Report menüü

Klõpsa ülaribalt vahekaarti Report. Siin asuvad kõik MS Projecti sisseehitatud aruanded, sh „Project Overview", „Cost Overview" ja „Work Overview".

2. Vali Project Overview

Klõpsa Dashboards → Project Overview. Aruanne avaneb uues vaates ja kuvab automaatselt projekti edenemise, verstapostid ja hilinenud ülesanded.

3. Project Overview ülevaade

Aruanne sisaldab nelja peamist osa:

  • % Complete – projekti üldine valmisolekuprotsent (suur oranž number)
  • Milestones Due – lähenevad verstapostid koos tähtaegadega
  • % Complete diagramm – tulpdiagramm kõikide tippülesannete kaupa
  • Late Tasks – tabel hilinenud ülesannetest

4. Kohanda diagrammi värve

Klõpsa tulpdiagrammil → ilmub paremal Chart Tools riba. Vali Design → Change Colors, et muuta tulpade värvipaletti. Konkreetse tulpa värvi muutmiseks topeltklõpsa sellel ja vali „Format Data Point".

5. Lõpptulemus

Valmis Project Overview aruanne: projekti valmisolekuprotsent, tulpdiagramm kõikide faasidega, verstapostide tabel ja hilinenud ülesannete nimekiri – kõik ühes vaates automaatselt uuenemas.
